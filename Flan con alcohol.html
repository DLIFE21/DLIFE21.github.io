💡 Concepto del Producto

Un flan tradicional con una cobertura tipo caramelo o mermelada de cereza (cherry), infusionada con una bebida alcohólica (como whisky, ron o vodka). Ideal para un público joven-adulto que busca sabores nuevos y sofisticados.

🥃 Recomendación de Alcohol

Para un perfil de sabor armónico, considera lo siguiente:

  1. Ron oscuro (tipo añejo)

    • ✅ Aporta notas cálidas, dulzonas, vainilla, especias.

    • 🔥 Combina muy bien con cereza y caramelo.

  2. Bourbon o whisky suave (como Jameson o Maker's Mark)

    • ✅ Le da profundidad, notas ahumadas, madera, miel.

    • 🥴 Evita los muy turbios o secos, que pueden chocar.

  3. Vodka saborizado (cherry, vainilla o neutro)

    • ✅ No altera mucho el sabor, ideal si quieres resaltar la fruta.

    • 💧 Neutral, pero puede faltar personalidad.

  4. Licor de cereza (Kirsch, Cherry Heering o maraschino)

    • ✅ Cereza pura + alcohol.

    • 💥 Ideal si quieres un sabor cherry intenso sin quemar.

📊 Porcentaje de Alcohol Recomendado

Tú no estás haciendo un cóctel, sino un postre gourmet, así que:

  • En la mezcla final del caramelo/mermelada:
    🔹 Entre 2% y 5% de alcohol real (después de cocción).
    🔹 Si reduces la mezcla a fuego lento, gran parte del alcohol se evapora, dejando solo el sabor.

  • Para mantener algo de efecto alcohólico (ligero):
    🔹 Agrega una pequeña parte de alcohol luego de retirar del fuego, para que no se evapore del todo.
    🔹 Ejemplo: 50 ml de alcohol por cada 500 ml de sirope.

🍒 ¿Cómo Preparar el Caramelo de Cherry con Alcohol?

Aquí un ejemplo práctico:

plaintext
Ingredientes:
- 1 taza de cerezas (pueden ser frescas o en conserva, sin hueso)
- 1/2 taza de azúcar morena
- 2 cdas de agua
- 1 cda de jugo de limón
- 50 ml de ron oscuro o licor de cereza

Preparación:

  1. Cocina las cerezas con azúcar, agua y limón a fuego medio-bajo.

  2. Cuando se forme una especie de sirope/mermelada, apaga el fuego.

  3. Agrega el alcohol (puede ser mitad antes y mitad después, para balancear sabor y retención alcohólica).

  4. Enfría y usa como cobertura sobre el flan.

🎯 Objetivo

Tener una página web estática (tipo vitrina digital) donde el usuario:

  1. Vea el producto (flan gourmet).

  2. Lea la descripción.

  3. Pueda “comprarlo” (simulación de compra con formulario).

  4. Acceda mediante código QR (enlace compartible).

⚙️ ¿Qué tecnologías usar?

Nada muy pesado, solo HTML, CSS y un poco de JS. Aquí el stack rápido:

  • Frontend: HTML5, CSS3, JavaScript

  • Hosting gratuito temporal: GitHub Pages o Netlify

  • Simulación de compra: Formulario tipo "Agregar al carrito" que al hacer clic confirme el pedido (puede ir a una página de “Gracias por tu pedido” o mostrarlo con JS).

  • QR Code: Usaremos una herramienta como qr-code-generator.com o el que trae tu navegador móvil para generar el QR una vez esté subida.

📦 Contenido de la Página

1. Inicio / Home

  • Foto del flan cherry con alcohol

  • Título: "Flan Flameado: Postre con Carácter"

  • Subtítulo: "La mezcla perfecta entre lo dulce y lo atrevido"

2. Descripción del Producto

  • Ingredientes

  • Descripción gourmet

  • Opción de selección (tamaño, cantidad, alcohol: ron, vodka…)

3. Formulario de Pedido (fake checkout)

  • Nombre

  • Teléfono / WhatsApp

  • Dirección

  • Cantidad

  • Opción de entrega: "Recoger" o "Delivery"

  • Botón: “Realizar pedido” (esto solo simula el proceso, muestra una alerta o lleva a una página de agradecimiento)

Aquí te dejo un ejemplo simplificado que puedes subir a GitHub Pages o Netlify y escanear con QR después:

<!DOCTYPE html>
<html lang="es">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1">
  <title>Flan Flameado</title>
  <style>
    body { font-family: Arial; padding: 20px; background: #fff6f6; color: #333; }
    .container { max-width: 600px; margin: auto; }
    h1 { color: #a10000; }
    img { width: 100%; border-radius: 8px; }
    button { background: #a10000; color: white; padding: 10px 20px; border: none; border-radius: 5px; cursor: pointer; }
    input, select { width: 100%; padding: 10px; margin: 5px 0; }
  </style>
</head>
<body>
  <div class="container">
    <h1>Flan Flameado 🍮🔥</h1>
    <img src="https://via.placeholder.com/600x400?text=Flan+Cherry+con+Alcohol" alt="Flan Gourmet">
    <p>Un flan artesanal con cobertura de cereza infusionada en licor. El balance perfecto entre dulzura y carácter.</p>

    <h3>Haz tu pedido</h3>
    <form id="orderForm">
      <input type="text" placeholder="Nombre completo" required>
      <input type="tel" placeholder="Teléfono o WhatsApp" required>
      <input type="text" placeholder="Dirección de entrega" required>
      <select required>
        <option disabled selected>Tipo de alcohol</option>
        <option>Ron oscuro</option>
        <option>Vodka</option>
        <option>Whisky</option>
      </select>
      <input type="number" placeholder="Cantidad" min="1" required>
      <select required>
        <option disabled selected>Modo de entrega</option>
        <option>Recoger en tienda</option>
        <option>Delivery</option>
      </select>
      <button type="submit">Realizar pedido</button>
    </form>

    <p id="confirmation" style="color: green; font-weight: bold;"></p>
  </div>

  <script>
    document.getElementById("orderForm").addEventListener("submit", function(e) {
      e.preventDefault();
      document.getElementById("confirmation").textContent = "¡Gracias por tu pedido! Nos pondremos en contacto pronto.";
    });
  </script>
</body>
</html>
